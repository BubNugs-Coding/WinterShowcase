# Winter Midterm Showcase (GitHub Pages)

This folder contains a **plain static GitHub Pages site** (an interactive “README-style” webpage).

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/` (SVG placeholders you can replace with real screenshots/GIFs)

## Publish to GitHub Pages (WinterShowcase repo)
1. Copy the contents of this folder into the root of your GitHub Pages repo:
   - `https://github.com/BubNugs-Coding/WinterShowcase`
2. Commit + push to the branch GitHub Pages is set to deploy from (usually `main`).
3. In GitHub repo settings: **Settings → Pages**
   - Build and deployment: **Deploy from a branch**
   - Branch: `main` and `/ (root)`

After it deploys, your site URL will typically be:
- `https://bubnugs-coding.github.io/WinterShowcase/`
